# ZCode - Интерпретируемый язык программирования на Java

ZCode - это собственный интерпретируемый язык программирования, написанный на Java. Проект включает полный стек: лексический анализатор, парсер, AST и интерпретатор.

## Особенности

- **Полный стек компилятора**: Лексический анализ, синтаксический анализ, построение AST
- **Богатый синтаксис**: Поддержка циклов, условий, функций, массивов
- **Встроенные функции**: Математические, строковые, файловые операции
- **Система типов**: Числа, строки, массивы
- **ООП дизайн**: Паттерн Visitor для обхода AST
